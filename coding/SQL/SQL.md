##### Data Types
* INT
* DECIMAL (M, N)
	* M: total digits
	* N: # of decimal digits
* VARCHAR(L)
	* String of text of length L
* BLOB
	* Binary Large Object, used to store large data
* DATE
	* YYYY-MM-DD
* TIMESTAMP
	* YYYY-MM-DD HH:MM:SS
#### Commands
**Creating a table:** 
``` plsql
CREATE TABLE student (
	student_id INT PRIMARY KEY
	name VARCHAR()
)
```
