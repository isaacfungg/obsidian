#### Decision Stumps
We learn a decision stump by finding the rule with the best score


#### Supervised Learning Notation
* $X_{ij}$ is feature 'j' for example 'i'
* $X_i$ is a list of all features for example 'i'
* $X^j$ is column 'j' of the matrix
Label vector 'y' contains the labels of the examples
* $y_i$ is the label of example 'i'
